Below is a complete 16-day structured plan for your April 4–30 camp (4 days/week, 90 mins each session). It keeps a consistent session structure so kids feel comfortable, while the skills progress week-by-week.

Age group: 7–15 years
Session time: 6:00–7:30 AM

Standard Daily Session Structure (Use Every Day)
Time	Activity
6:00–6:10	Warm-up games
6:10–6:25	Movement / agility drills
6:25–6:45	Skill drills
6:45–7:05	Game-based drills
7:05–7:20	Mini match / play
7:20–7:25	Cool down
7:25–7:30	Question of the day
